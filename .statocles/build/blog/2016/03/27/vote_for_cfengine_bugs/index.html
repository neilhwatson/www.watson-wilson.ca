Vote for your favourite CFEngine bugs

Tags: cfengine configuration management

Posted on 2016-03-27

CFEngine has switched to a JIRA bug system and this system allows us to vote for bugs. I encourage you to vote for your favourite bugs; here are mine:

  1. 2033 YAML data files are not parsed correctly. I hope to make EFL work with YAML data files.
  2. 85 and 86 involve cf-agent's inability to log promise outcomes.
  3. 1520 is a whole list of IPv6 related bugs. While CFEngine does work with IPv6, IPv6 is a second class citizen.
